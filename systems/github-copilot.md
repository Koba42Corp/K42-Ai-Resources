# GitHub Copilot

## Overview

GitHub Copilot is an AI-powered code completion tool that suggests code as you type. It's powered by OpenAI's Codex model and integrates directly into your IDE to provide context-aware code suggestions.

**Type**: AI Coding Assistant / IDE Extension  
**License**: Proprietary (Subscription)  
**Primary Language**: Multiple (50+ languages)  
**Founded**: 2021  
**Website**: [github.com/features/copilot](https://github.com/features/copilot)

---

## What is GitHub Copilot?

GitHub Copilot is an AI pair programmer that suggests code completions, functions, and entire code blocks based on your comments and existing code. It integrates with popular IDEs and provides real-time suggestions.

### Key Characteristics

- **AI-Powered**: Powered by OpenAI Codex
- **IDE Integration**: VS Code, JetBrains, Neovim
- **Multi-Language**: Supports 50+ languages
- **Context-Aware**: Understands your codebase
- **Real-Time**: Suggestions as you type
- **Code Generation**: Generates entire functions

## Core Features

### Code Completion

```javascript
// Type a comment
// Function to calculate fibonacci

// Copilot suggests:
function fibonacci(n) {
  if (n <= 1) return n;
  return fibonacci(n - 1) + fibonacci(n - 2);
}
```

### Multi-File Context

- Understands your codebase
- Suggests based on project structure
- Learns from your code style

### Language Support

- JavaScript/TypeScript
- Python
- Go, Rust, Java
- And 50+ more languages
